Stack Tecnológico Completo

Capa	Tecnología
- Backend API	FastAPI
- Frontend/Dashboard	Streamlit + Plotly
- Base de Datos	PostgreSQL
- ORM	SQLAlchemy + Pydantic
- Visión Artificial	YOLOv8 (Ultralytics)
- ML/ Predicciones	Scikit-learn
- Entorno Virtual	venv
- Contenedor	Docker (futuro)

## Resumen del Proyecto

- Backend: FastAPI puro
- Frontend: Streamlit (Dashboard interactivo)
- Datos: PostgreSQL (tú introduces tus productos)
- IA: YOLOv8local (detecta productos en fotos de estanterías)
- ML: Scikit-learn (predice cuándo se acabará el stock)
